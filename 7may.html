User interface intro notes


What is a user interface (UI)? :-

The user interface (UI) is the point of human-computer interaction and communication in a device. This can include display screens, keyboards, a mouse and the appearance of a desktop. It is also how a user interacts with an application or a website, using visual and audio elements, such as type fonts, icons, buttons, animations and sounds.

Parts of User interface Design:-

The fundamental parts of most UIs are as follows:

> Input controls

> Navigational components

> Informational components

> Containers

Input controls:-

An input control defines the way in which the system captures information and it is the most itneractive component of the UI. Some common interface input controls consist of elements such as buttons, radio buttons, check boxes, drop-down buttons, list boxes and text fields.

Here is a list of the input control elements

1 Butons

2 Radio butons

3 Check box

4 Drop-down list

5 Drop-down button

6 List box

7 Text field

Navigational components:-

Navigational components provide a way for the users to tell the system what to do. Some of these includes search fields, breadcumbs, pagination, tags, icons and image carousel.

list of the navigational components elements

1 Search Field

2 Breadcumb

3 Pagination

4 Tags

5 Icon

6 Image Carousel
